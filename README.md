# Tuzi

Tuzi is a terminal text editor written in MoonBit.

To play with it:

```
moon run src/main --target native -- path/to/file
```

## Key bindings

- `Ctrl-S` to save
- `Ctrl-Q` to quit
- `Ctrl-N` to next line
- `Ctrl-P` to previous line
- `Ctrl-F` to forward
- `Ctrl-B` to backward
- `Ctrl-A` to home
- `Ctrl-E` to end
- more to come

## Features

- syntax highlighting for .mbt files
